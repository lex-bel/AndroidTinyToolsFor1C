# AndroidTinyToolsFor1C

Набор дополнительных функций для мобильного приложения 1С под Android.

Сейчас компонента поддерживает:

- получение `Android ID` устройства;
- сканирование штрихкодов камерой устройства;
- работу с Bluetooth-сканером штрихкодов;
- подписку на широковещательные сообщения Android;
- вибрацию;
- короткий звуковой сигнал;
- `Toast`-сообщения.

## Методы компоненты

### 1. Получение идентификатора устройства

`GetDeviceId()`

Возвращает `Android ID` текущего устройства.

Пример:

```bsl
Перем AndroidTools;

Процедура ПриОткрытии()
    AndroidTools = Новый("AddIn.AndroidTinyTools");
    Сообщить("DeviceId: " + AndroidTools.GetDeviceId());
КонецПроцедуры
```

### 2. Сканирование камерой устройства

`StartCameraBarcodeScanner([Подсветка], [РежимСканирования], [ШиринаОбластиПроц], [ВысотаОбластиПроц], [ЦифровойЗумПроц])`

Открывает встроенный сканер с использованием камеры устройства. Поддерживаются линейные штрихкоды, `QR` и `Data Matrix`.

После успешного сканирования генерируется внешнее событие:

- Источник: `AndroidTinyTools`
- Событие: `Barcode`
- Данные: считанное значение кода

Дополнительно:

- на экране показывается рамка области распознавания;
- вне рамки область затемняется;
- в окне сканера есть кнопка ручного включения/выключения фонарика.

Параметры:

- `Подсветка` — `Истина/Ложь`, включает фонарик сразу при запуске.
- `РежимСканирования`:
  - `0` — универсальный режим;
  - `1` — быстрый режим для линейных 1D-штрихкодов;
  - `2` — только `QR`;
  - `3` — только `Data Matrix`;
  - `4` — `QR + Data Matrix`.
- `ШиринаОбластиПроц` — ширина центральной области распознавания в процентах экрана.
- `ВысотаОбластиПроц` — высота центральной области распознавания в процентах экрана.
- `ЦифровойЗумПроц` — цифровой зум `0..100` через сужение центральной области распознавания. `0` — без дополнительного зума, `100` — максимально узкий центральный crop.
Значения по умолчанию:

- если параметры области не заданы, используются встроенные пресеты под тип сканирования;

`StopCameraBarcodeScanner()`

Принудительно закрывает окно сканирования.

Базовый пример:

```bsl
Перем AndroidTools;

Процедура ПриОткрытии()
    AndroidTools = Новый("AddIn.AndroidTinyTools");
КонецПроцедуры

Процедура СканироватьКомандой(Команда)
    AndroidTools.StartCameraBarcodeScanner();
КонецПроцедуры

Процедура ВнешнееСобытие(Источник, Событие, Данные)
    Если Источник = "AndroidTinyTools" И Событие = "Barcode" Тогда
        Сообщить("Сканировано: " + Данные);
    КонецЕсли;
КонецПроцедуры
```

Примеры рекомендуемых режимов:

```bsl
// Быстрый режим для обычных линейных штрихкодов
AndroidTools.StartCameraBarcodeScanner(Ложь, 1, 80, 25);

// Только Data Matrix с подсветкой
AndroidTools.StartCameraBarcodeScanner(Истина, 3, 55, 55);

// QR + Data Matrix с ограничением центральной области
AndroidTools.StartCameraBarcodeScanner(Истина, 4, 60, 55);

// Data Matrix с цифровым зумом через центральный crop
AndroidTools.StartCameraBarcodeScanner(Истина, 3, 55, 55, 45);

// Закрыть сканер вручную
AndroidTools.StopCameraBarcodeScanner();
```

Замечания по производительности:

- для линейных кодов лучше использовать режим `1`;
- для маркировки `Data Matrix` лучше использовать режим `3`;
- слишком маленькая область распознавания ускоряет поиск, но пользователю сложнее попасть кодом в рамку.
- цифровой зум здесь программный: он не меняет оптику камеры, а сужает центральную область декодирования.

### 3. Работа с Bluetooth-сканером

`GetBluetoothDevicesList()`

Возвращает строку со списком сопряжённых Bluetooth-устройств в формате:

```text
ИмяУстройства///MAC-адрес
```

Каждое устройство отделяется переводом строки.

`StartBluetoothScannerHandler(<MAC-адрес>)`

Запускает обработчик сообщений от Bluetooth-сканера.

`StopBluetoothScannerHandler()`

Останавливает обработчик сообщений от Bluetooth-сканера.

`IsBluetoothScannerHandlerConnected()`

Возвращает `Истина`, если подключение к Bluetooth-сканеру установлено.

Пример:

```bsl
Перем AndroidTools;

Процедура ПриОткрытии()
    AndroidTools = Новый("AddIn.AndroidTinyTools");
КонецПроцедуры

Процедура ПодключитьСканер(Команда)
    СписокУстройств = AndroidTools.GetBluetoothDevicesList();
    МассивСтрок = СтрРазделить(СписокУстройств, Символы.ПС);

    Если МассивСтрок.Количество() = 0 Тогда
        Сообщить("Bluetooth-устройства не найдены");
        Возврат;
    КонецЕсли;

    ДанныеУстройства = СтрРазделить(МассивСтрок[0], "///");
    MACАдрес = ДанныеУстройства[1];
    AndroidTools.StartBluetoothScannerHandler(MACАдрес);
КонецПроцедуры

Процедура ВнешнееСобытие(Источник, Событие, Данные)
    Если Источник = "AndroidTinyTools" И Событие = "Barcode" Тогда
        Сообщить("Сканировано: " + Данные);
    КонецЕсли;
КонецПроцедуры
```

### 4. Подписка на broadcast-сообщения Android

`StartBroadcastReceiver(<ИмяСобытия>, <ИмяПараметра>)`

При получении сообщения вызывает обработчик `ВнешнееСобытие`.

Пример:

```bsl
Перем AndroidTools;

Процедура ПриОткрытии()
    AndroidTools = Новый("AddIn.AndroidTinyTools");
    AndroidTools.StartBroadcastReceiver("com.google.android.c2dm.intent.RECEIVE", "data");
КонецПроцедуры
```

Важно: переменная с объектом компоненты должна оставаться доступной, пока подписка нужна приложению.

### 5. Вибрация

`Vibrate([<ДлительностьВМиллисекундах>])`

По умолчанию: `500`.

Пример:

```bsl
AndroidTools.Vibrate(300);
```

### 6. Звуковой сигнал

`Beep([<КодСигнала>])`

По умолчанию используется код `28`.

Пример:

```bsl
AndroidTools.Beep();
AndroidTools.Beep(24);
```

### 7. Toast-сообщение

`Toast(<Текст>)`

Показывает стандартное всплывающее сообщение Android.

Пример:

```bsl
AndroidTools.Toast("Привет!");
```

## Сборка

Для сборки используется:

- Android Gradle Plugin `8.9.1`
- Gradle Wrapper `8.11.1`
- `compileSdkVersion 34`
- `targetSdkVersion 34`

Пример команды:

```powershell
.\gradlew.bat :app:assembleDebug
```

Артефакты упаковки складываются в каталог `AndroidTools`.

Для release-сборки можно использовать:

```powershell
.\gradlew.bat :app:assembleRelease
```

После `assembleRelease` в `AndroidTools` автоматически создаётся ZIP-пакет расширения 1С вида `AndroidTinyTools_<версия>.zip`.
